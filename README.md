# 商品交易可视化仪表盘

这是一个完全本地运行的静态仪表盘，用于分析近三年商品交易数据。打开 `index.html` 后导入 CSV 即可查看交易量、交易价格、交易额、商品贡献、价格分布、量价关系、月度季节性、异常波动和数据质量检查。

## 功能

- CSV 导入，支持 UTF-8 和常见中文 Windows 编码自动解码。
- 日期、商品、交易量、交易价格、交易金额字段自动识别，也支持手动映射。
- 日期范围、商品、多周期聚合和价格均线筛选。
- KPI 总览：交易量、交易额、加权均价、商品数、价格波动率、数据有效率。
- 图表：量价双轴趋势、交易额走势、商品贡献排行、价格分布、量价散点、月度热力图。
- 表格：异常波动、数据质量检查、明细预览。
- 导出：单个图表 SVG、周期汇总 CSV。

## 数据安全

- 不依赖 CDN 或第三方脚本。
- 不上传文件，不请求外部接口。
- 不使用 `localStorage` 保存数据。
- CSV 只通过浏览器 File API 读入当前页面内存。
- 关闭页面或点击“清空”后，页面内的数据即被释放。
- 仓库默认通过 `.gitignore` 忽略 `.csv`、`.xlsx`、`.docx`、`.zip` 等本地数据和材料。

## CSV 字段

仪表盘会自动识别常见中英文表头，也可以在页面里手动改字段映射。

推荐字段：

```csv
交易日期,商品名称,成交量,成交单价,成交金额
2024-01-08,冷轧钢卷,120,5180,621600
```

`成交金额` 可选；如果没有金额列，会自动用 `成交量 * 成交单价` 计算。

## 使用

直接打开：

```bash
open index.html
```

如果浏览器不允许直接打开本地文件，也可以在当前目录启动一个本地服务：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

也可以用公开示例数据测试：

```bash
examples/sample.csv
```

## Excel 转 CSV

如果原始文件是 `.xlsx`，可以使用 `tools/xlsx_to_csv.py` 在本机转换：

```bash
python3 tools/xlsx_to_csv.py input.xlsx output.csv
```

转换脚本只读取第一个工作表，适合常规明细表。复杂合并单元格、公式未缓存值、隐藏工作表等情况建议先在 Excel 里另存为 CSV。
